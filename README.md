## Getting Started
### Prerequisites
* Python 3.10.7
* XAMPP

### Installation

* ```git clone https://github.com/LiamYS/showering-app.git```
* ```cd onlinekozijnen```
* ```py -3 venv venv```
* ```venv\Scripts\activate```
* ```pip install Flask```
* ```pip install flask_mysqldb```
* ```npm install```
